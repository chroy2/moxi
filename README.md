# Moxi

Follows the [tutorial](https://youtu.be/vT4-bvHCbE0?si=VE9nA9tVZa4WpRfT) for microbit by [TheRustyBits](https://github.com/therustybits)

Air quality, Temperature, and Humidity monitor.
Thingy32 App integration

## Hardware

- [Microbit](https://microbit.org/buy/bbc-microbit-single/)
- [BitMaker V2](https://www.seeedstudio.com/CH-BitMaker-V2-p-5330.html)
- [SCD 41](https://www.adafruit.com/product/5190) or [SCD 40](https://www.adafruit.com/product/5187)
- [Groove to STEMMA QT / Qwiic Cable](https://www.adafruit.com/product/4424)
